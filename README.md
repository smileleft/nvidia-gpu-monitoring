# nvidia-gpu-monitoring
repository for nvidia gpu monitoring environment

## nvidia dcgm(Data Center GPU Manager) build
```
# get dcgm git src
git clone https://github.com/NVIDIA/dcgm-exporter.git

# move directory
cd ./dcgmbuild

# run build script
./build.sh
```
